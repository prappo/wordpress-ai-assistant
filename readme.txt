A powerful WordPress AI assistant that integrates multiple AI models to help with content generation, analysis, and website management.

== Description ==

WordPress AI Assistant is a comprehensive AI-powered plugin that brings the power of multiple leading AI models directly into your WordPress dashboard. With support for various AI providers and a rich set of features, it helps streamline your content creation and website management tasks.

= Key Features =

**Multiple AI Models Support**
* OpenAI Models (GPT-4, GPT-3.5 Turbo)
* Anthropic Models (Claude 3 Opus, Claude 3 Sonnet, Claude 3 Haiku)
* Google Models (Gemini Pro)
* Groq Models (Mixtral, LLaMA, Gemma, DeepSeek variants)
* DeepSeek Models (DeepSeek Chat, DeepSeek Reasoner)
* OpenRouter Integration for open-source models

**Customization**
* Flexible theme options (Light, Dark, System)
* Customizable system prompts
* Configurable AI agent settings
* Provider-specific configurations

**User Interface**
* Modern and intuitive chat interface
* Thread management system
* Real-time response streaming
* Fully responsive design
* Clean and professional look

**Security & Integration**
* Secure API key management
* Seamless WordPress integration
* Role-based access control
* Data privacy compliance

**Developer Features**
* Extensible architecture
* Tool integration support
* Comprehensive error handling
* Performance optimized

= Requirements =

* WordPress 5.9 or higher
* PHP 7.2 or higher
* API key from at least one supported provider

= Installation =

1. Upload the plugin files to the `/wp-content/plugins/wordpress-ai-assistant` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->AI Assistant screen to configure the plugin
4. Enter your API keys for desired providers
5. Start using the AI assistant from your WordPress dashboard

= Configuration =

1. Navigate to the plugin settings page
2. Enter your API keys for desired providers
3. Customize system prompts and agent settings
4. Choose your preferred theme
5. Save changes and start using the assistant

== Frequently Asked Questions ==

= Which AI models are supported? =

The plugin supports multiple AI models from various providers including OpenAI, Anthropic, Google, Groq, DeepSeek, and models available through OpenRouter.

= Do I need API keys for all providers? =

No, you only need an API key for the provider(s) you wish to use. You can start with one provider and add others later.

= Is it compatible with my theme? =

Yes, the plugin is designed to work with any WordPress theme and includes its own theme system for the assistant interface.

= How secure are my API keys? =

API keys are stored securely in the WordPress database using WordPress security best practices.
